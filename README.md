# How to Use Password Storage
- - - - - - - - - - - - - - 

## Setup

Simply install via `npm`.

```bash
npm install -g pwds
```

## Display passwords

Go to a directory where you want to save the encrypted password file.

Then do:

```bash
pw-list [filename] [backup_dir]
```
... and enter the password.


## Add passwords

```bash
./pw-add
```

Follow the prompts on the screen.

## Remove a password

```bash
./pw-remove
```

Follow the prompts on the screen.
